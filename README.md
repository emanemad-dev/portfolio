# Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML, CSS, JavaScript, and TailwindCSS**.  
This project showcases a **custom portfolio**, **testimonials carousel**, **dark/light mode toggle**, **scroll-to-top button**, and **customizable sidebar** for changing colors and fonts.

## üéØ Features

- **Responsive Navbar with Active Link**: Updates automatically on scroll.  
- **Dark/Light Theme Toggle**: Switch between dark and light modes.  
- **Portfolio Section with Filters**: Show different categories using JavaScript tabs.  
- **Testimonials Carousel**: Smooth horizontal scrolling with active indicators.  
- **Settings Sidebar**: Customize theme colors and fonts.  
- **Scroll to Top Button**: Smooth scroll back to the top.  
- **LocalStorage Persistence**: Saves theme color, font, and mode preferences.
